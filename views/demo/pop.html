* 弹出框
* @param {string} msg 弹出框显未内容
* @param {mixed} callback 点击确认按钮后的回调函数，可以是函数， 也可以是可执行的js代码串
* @param {int} width 宽度
* @param {string} title 窗口的标志
* @param {string} btn_text 按钮文字
* @param {int} icon 图标( 0 -8 )
